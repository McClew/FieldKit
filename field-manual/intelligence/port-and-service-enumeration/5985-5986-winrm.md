# 5985, 5986 - WinRM

## Port 5985 & 5986

WinRM (Windows Remote Management) uses default TCP ports 5985 for HTTP and 5986 for HTTPS.

## Configuration Check

Check WinRM configuration and service status.

{% code title="Check if WinRM is running (from target)" %}
```powershell
Get-Service WinRM
```
{% endcode %}

{% code title="Check WinRM configuration" %}
```bash
winrm get winrm/config
```
{% endcode %}

{% code title="Check listeners" %}
```bash
winrm enumerate winrm/config/listener
```
{% endcode %}

{% code title="Test if WinRM is accessible from remote" %}
```powershell
Test-WSMan -ComputerName target.com
```
{% endcode %}
