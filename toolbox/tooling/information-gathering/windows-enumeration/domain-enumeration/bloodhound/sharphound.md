# SharpHound

> SharpHound is the official data collector for BloodHound. It is written in C# and uses native Windows API functions and LDAP namespace functions to collect data from domain controllers and domain-joined Windows systems.

{% hint style="info" %}
#### Download & Install

Install via apt:

`sudo apt install sharphound`
{% endhint %}

***

## Command

```bash
sharphound -h
```

```
> sharphound ~ for BloodHound CE collector

/usr/share/sharphound
|-- SharpHound.exe
|-- SharpHound.exe.config
|-- SharpHound.pdb
`-- SharpHound.ps1
```
